# ntc telegram client in terminal (cli)

## fully open source telegram client in terminal made with ai

python packages:
```
pip install telethon python-dotenv
```

Telegram client in terminal with lot of commands
to start u need create .env file in folder with .py file
.env file:
```
API_ID=
API_HASH=
```
get API_ID and API_HASH at **[my.telegram.org](my.telegram.org)**

to start using ntc type ``` ntc --help ```
